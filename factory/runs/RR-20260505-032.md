# Run Record: Add Encrypted Evidence Export

## Assignment

Add encrypted local decision-evidence export for human review.

## What Changed

- Added decision evidence payload construction with safe decision fields,
  signals, feature values, versions, trace IDs, and no-filing metadata.
- Added AES-256-GCM evidence encryption with Scrypt-derived passphrase keys.
- Added evidence decryption support for local tests and verification.
- Added `fraud-v2 evidence-export` with passphrase read from
  `FRAUD_EVIDENCE_PASSPHRASE` by default.
- Added unit coverage for round trip decrypt, short passphrase rejection, and
  wrong passphrase rejection.
- Updated docs, PR draft, and version metadata to `0.32.0`.
