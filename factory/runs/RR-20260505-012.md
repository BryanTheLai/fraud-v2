# Run Record: Local Tracing And Alerts

## Assignment

Add local request tracing, structured request logs, HTTP metrics, and alert rules
without introducing cloud dependencies.

## What Changed

- Added `fraud_v2.observability.logging`.
- Added request middleware that:
  - accepts `X-Request-ID`
  - returns `X-Trace-ID`
  - writes structured JSON request logs
  - emits HTTP request counters and latency histograms
- Added Prometheus alert rules for:
  - API unavailable
  - decision p95 latency above 500ms
  - HTTP 5xx responses
- Mounted alert rules into the Docker full profile.
- Expanded full smoke to verify Prometheus rule loading.
- Added tests for trace headers, request metrics, and alert rule files.
- Updated docs and PR draft.
- Bumped package/API version to `0.12.0`.
