# Run Record: Model Registry Controls

## Assignment

Add local model governance with shadow/active controls.

## What Changed

- Added `RegisteredModel`, `ModelStatus`, and `JsonModelRegistry`.
- Registry tracks model/report hashes, feature columns, metrics, thresholds,
  status, registration time, and notes.
- Added `fraud-v2 model-register`, `model-list`, and `model-promote`.
- Promotion enforces one active model by demoting any prior active model to
  shadow.
- Bumped package/API version to `0.6.0`.
- Added unit tests for registration, promotion, and missing model handling.
